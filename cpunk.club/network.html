This guide shows how to add the CPUNK network to your Cellframe node.

Download and Install Files

1
Network Configuration
Download Cpunk.cfg
üêß
Linux: Copy to /opt/cellframe-node/etc/network/Cpunk.cfg 
sudo cp Cpunk.cfg /opt/cellframe-node/etc/network/Cpunk.cfg
ü™ü
Windows: Copy to C:\Users\Public\Documents\cellframe-node\etc\network\Cpunk.cfg 
copy Cpunk.cfg C:\Users\Public\Documents\cellframe-node\etc\network\Cpunk.cfg
2
Chain Configuration Files
Download chain-0.cfg Download main.cfg
üêß
Linux: Create directory and copy files 
sudo mkdir -p /opt/cellframe-node/etc/network/Cpunk
sudo cp chain-0.cfg /opt/cellframe-node/etc/network/Cpunk/
sudo cp main.cfg /opt/cellframe-node/etc/network/Cpunk/
ü™ü
Windows: Create directory and copy files 
mkdir C:\Users\Public\Documents\cellframe-node\etc\network\Cpunk
copy chain-0.cfg C:\Users\Public\Documents\cellframe-node\etc\network\Cpunk\
copy main.cfg C:\Users\Public\Documents\cellframe-node\etc\network\Cpunk\
3
Root Certificates
cpunk.root.0.dcert cpunk.root.1.dcert cpunk.root.2.dcert cpunk.master.0.dcert cpunk.master.1.dcert cpunk.master.2.dcert
üêß
Linux: Copy to /opt/cellframe-node/share/ca/ 
sudo cp cpunk.root.0.dcert cpunk.root.1.dcert cpunk.root.2.dcert cpunk.master.0.dcert cpunk.master.1.dcert cpunk.master.2.dcert /opt/cellframe-node/share/ca/
ü™ü
Windows: Copy to C:\Users\Public\Documents\cellframe-node\share\ca\ 
copy cpunk.root.0.dcert cpunk.root.1.dcert cpunk.root.2.dcert cpunk.master.0.dcert cpunk.master.1.dcert cpunk.master.2.dcert C:\Users\Public\Documents\cellframe-node\share\ca\
4
Restart Node and Verify
üêß
Linux: Restart and verify 
sudo systemctl restart cellframe-node
/opt/cellframe-node/bin/cellframe-node-cli net list
ü™ü
Windows: Restart and verify 
Restart-Service -Name "Cellframe Node" -Force
& "C:\Program Files\cellframe-node\cellframe-node-cli.exe" net list
